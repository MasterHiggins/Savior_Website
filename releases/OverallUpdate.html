Beta Release Event

Article from Chris Higgins
(Developer)
This article does not contain spoilers

Hello everyone.

No huge update trailer today, but because I usually do an update during the Spring, I wanted to give one now to manage expectations beforehand.

At this point, development on Part 1 has essentially concluded. I consider this part finished, and Part 2 has begun loosely. Some friend have begun testing the full game, and a majority of game breaking bugs are fixed. We are on Release candidate 4.

If there is ever an issue you come across when playing the game on May 22nd, 2026, shoot me a message and I will, hopefully, fix your save file and fix the bug. However, if it's small, I have also created a save file editor so you can edit it yourself, or you can just fool around with it. See it here.

Also, I have made this countdown until the game comes out, which is linked in the button below, as is everything else worthwhile from above.

Anyway, as for the updates for Part 2, I want them to be great, grand, all that fun stuff. Larger presentations, better trailers, etc. Because of this, the the regular updates I do will likely be a much different schedule. I'll still try to give a progress update every season, but trailers will be much more random.

Lastly, the steam page is in progress. Currently doing all the forms and whatnot, but it should be up soon with preorders immediately when I can. Hopefully soon.

I likely won't have another update until the Steam page is up.

Thank you.
