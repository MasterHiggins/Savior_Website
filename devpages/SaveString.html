Save String Contents

Please only directly modify the save string when absolutely necessary.

Example String: "1,lava:0:0,5,216,1369,180,5,1,0,0,false,"

    The first value is the weapon level.
        The second value is Area ID.
        The third value is the Player's Health.
        The fourth value is the Player's X Position.
        The fifth value is the Player's Y Position.
        The sixth value is the Player's Direction.
        The seventh value is the Player's Maximum Health.
        The eight value is the Current Quest.
        The ninth value is the Player's Material Count.
        The tenth value is the Map Change.
        The eleventh value is the Spells Unlocked Boolean Value.

      The reason why there is a comma at the end is because the game reads that as the last value in the string and stops checking from there.
